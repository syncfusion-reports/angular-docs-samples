Syncfusion Angular Reports docs samples repository contains the  the `Report Viewer` and `Report Designer` samples. It empower your web application with feature-rich report preview, edit, and customization capabilities that allow you to explore data easily and make real-time decisions. Effortlessly print and export reports.

## Requirements to run the demo

The samples requires the below requirements to run.

* [Node.js](https://nodejs.org)

## Using the Reporting Samples

### Installing

To install all dependent packages, use the below command

```bash
npm install
```

### Building

To compile the source files, use the below command

```bash
npm run build
```

### Running

To run the samples, use the below command

```bash
npm run serve
```
